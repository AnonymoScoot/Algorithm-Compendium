# Algorithm Compendium

**Algorithm Compendium** is a polyglot collection of algorithms and data structures implemented in multiple programming languages.  
The goal of this repository is to serve as a personal archive, learning resource, and reference implementation hub for core computer science concepts.


## Repository Structure
The repository is organized by programming language, topic, and algorithm:

```
Algorithm-Compendium/
├── python/
│ ├── strings/
│ │ └── palindrome/
│ ├── sorting/
│ │ └── quick_sort/
│ └── graphs/
│ └── bfs/
├── cpp/
│ └── sorting/
│ └── merge_sort/
└── java/
```

Each directory may contain implementations in languages such as:
- Python
- C++
- Java
- JavaScript
- Or others as added over time

## Contributions

Contributions are welcome!

If you want to add an algorithm:

1. Fork the repository
2. Create a new branch
3. Add your implementation following existing style conventions
4. Submit a pull request

Please ensure code is:

- Well-documented
- Readable
- Correctly tested if possible
